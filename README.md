# 💰 Controle Financeiro Pessoal

Projeto desenvolvido para fins acadêmicos como parte das atividades do curso de **Análise e Desenvolvimento de Sistemas (ADS)** da **FIAP**.

## 📖 Descrição

O **Controle Financeiro Pessoal MyFin** é uma aplicação web que tem como objetivo auxiliar usuários no acompanhamento e gerenciamento de suas finanças. A solução oferece uma interface simples e intuitiva para visualização de receitas, despesas e saldo disponível, promovendo maior organização financeira.

Este projeto foi desenvolvido aplicando conceitos de desenvolvimento web front-end, usabilidade e organização de informações, demonstrando na prática os conhecimentos adquiridos durante a graduação.

## 🎯 Objetivos

* Facilitar o controle financeiro pessoal;
* Permitir a visualização organizada de receitas e despesas;
* Apresentar indicadores financeiros de forma clara;
* Aplicar conceitos de HTML, CSS e JavaScript;
* Desenvolver uma interface estática, porém moderna e responsiva.

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3 Tailwind
* JavaScript

## 💻 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/flealbrl/myfin-visao-geral
```

2. Acesse o diretório do projeto:

```bash
cd /myfin-visao-geral
```

3. Abra o arquivo `index.html` em seu navegador de preferência.

Não é necessária a instalação de dependências adicionais.
